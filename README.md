# :tooth: АИС Регистратура Стоматологии
![Логотип](WindowsFormsApp1/Resources/logo.png)
Добро пожаловать в проект **Автоматизированная информационная система по ведению регистратуры в стоматологии на базе платформы .NET**.  
Данное приложение разработано на **C# (WinForms)** и предназначено для автоматизации работы регистратуры стоматологии:  
ведение пациентов, запись на приём, учёт врачей и расписания.

---
## :clipboard: Описание проекта

**АИС Регистратора Стоматологии** помогает облегчить работу персонала клиники, обеспечивая:
- :page_facing_up: **Хранение информации о пациентах, врачах**  
- :calendar: **Планирование приёмов и расписания врачей**  
- :man_health_worker: **Учёт талонов**  
- :bar_chart: **Создание отчётов**
